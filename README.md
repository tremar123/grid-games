GridGames is a flask web application where you can sign up and play few games that are based on grid.

* [Whac-a-mole](#whac-a-mole)
* [Connect four](#connect-four)
* [Snake](#snake)
* [Space Invaders](#space-invaders)
* [Frogger](#frogger)
* [Tetris](#tetris)
