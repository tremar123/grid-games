# games-web
#### Video Demo:  <https://www.youtube.com/watch?v=-ttqQJfbDzA>
#### Description:
GridGames is a flask web application where you can sign up and play few games that are based on grid.

These games include:
* [Memory game](#memory-game)
* Whac-a-mole
* Connect four
* Snake
* Space Invaders
* Frogger
* Tetris

## Memory game
